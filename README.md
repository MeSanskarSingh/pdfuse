# 📄 PDFuse

A clean, fast, and minimal web app to merge multiple PDF files into a single document — directly in the browser.

## Features

* Upload multiple PDF files
* Merge PDFs instantly
* Client-side processing (no server required)
* Minimal and distraction-free UI

## Tech Stack

* Next.js (Pages Router)
* Tailwind CSS v4
* JavaScript

## Libraries Used

* pdf-lib → For PDF manipulation and merging
* react-dropzone → For drag-and-drop file uploads
* react-icons → For UI icons

## How It Works

1. User uploads multiple PDF files
2. Files are processed in the browser
3. Pages are extracted and merged using pdf-lib
4. A new PDF is generated and downloaded instantly

## Privacy

All processing happens locally in your browser.
No files are uploaded to any server.
